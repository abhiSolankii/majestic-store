# MajesticStore

MajesticStore is an e-commerce web application built with React and Vite. It allows users to browse products, view product details, add items to the cart, and manage their shopping cart. The application uses local storage to persist cart data across page refreshes.

## Features

- Browse products with search and filter options
- View detailed information about each product
- Add products to the shopping cart
- Update product quantities in the cart
- Remove products from the cart
- Persist cart data using local storage
- Responsive design with Tailwind CSS

## Technologies Used

- React: A JavaScript library for building user interfaces
- Vite: A fast build tool for modern web projects
- React Router: Declarative routing for React applications
- Axios: Promise-based HTTP client for making API requests
- Tailwind CSS: A utility-first CSS framework for rapid UI development
- React Icons: Popular icons as React components
- React Hot Toast: Toast notifications for React applications

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm (version 6 or higher) or yarn

### Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/your-username/majesticstore.git
   cd majesticstore
   ```

2. Install dependencies:

npm install

3. Running the Application

To start the development server:

npm run dev

The application will be available at http://localhost:3000.

4. Building for Production
   To build the application for production:

The production-ready files will be in the dist directory.

### API

- The application uses the Fake Store API to fetch product data.

- Custom Hooks
- useApiRequest: A custom hook for making API requests using Axios.
- Context
- CartContext: Provides cart state and actions to add, update, and remove items from the cart.
